# COS30043_Final_Project

## Project Overview

This project is developed for COS30043 - Interface Design and Development.

The application is being built using:

- Vue 3
- Vite
- Vue Router
- Bootstrap 5

## Project Theme

GameHub - Gaming Community Platform

Users can:

- Browse games
- Read gaming news
- Write reviews
- Create discussion posts
- Like content
- Manage their own content

## Planned Features

### Stage 1
- Home Page

- About Page
- Search functionality
- Pagination
- Responsive design

### Stage 2
- User registration
